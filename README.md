# Udaya Landing

Текущая версия проекта: статический лендинг для `Udaya` с одной входной страницей [`index.html`](./index.html).

Что уже изменено:

- из первого блока убрана правая колонка, чтобы герой был чище и спокойнее;
- мятный акцент сделан ярче;
- важные, узнаваемые для пользователя фразы подсвечены мятным только в заголовках;
- мобильная версия ужата и выровнена: компактнее шапка, спокойнее интервалы, лучше читаются заголовки и карточки.
- зеркало сайта поднято на `Vercel`: [site-gamma-nine-66.vercel.app](https://site-gamma-nine-66.vercel.app).

## Стек

- `HTML + CSS`
- `serve` для локального запуска и Railway
- smoke-тест на `Node.js`

## Требования

- `Node.js >= 20`
- `npm >= 10`

## Установка зависимостей

```bash
npm install
```

## Переменные окружения

Скопируйте пример и при необходимости поменяйте порт:

```bash
cp .env.example .env
```

Доступные переменные:

- `PORT` — порт локального или Railway-запуска. По умолчанию `3000`.

## Локальный запуск

Режим разработки:

```bash
npm run dev
```

Прод-режим локально:

```bash
npm start
```

## Тесты

Smoke-проверка поднимает локальный сервер и проверяет:

- страница открывается по HTTP;
- герой рендерится;
- правая часть первого блока действительно убрана;
- мятные акценты в ключевых заголовках попали в HTML.

Запуск:

```bash
npm test
```

## Деплой в Railway

Первичный деплой:

```bash
railway login
railway init
railway up
```

Повторный деплой после изменений:

```bash
railway up
```

Если проект уже привязан к сервису, Railway подхватит `npm run start` из [`railway.json`](./railway.json).

## Что проверить после деплоя

Технически:

- главная страница открывается без `404`;
- шрифты грузятся;
- адаптив на ширине `390px` и `430px` не ломает сетку;
- в первом экране больше нет правой карточки.

Продуктово:

- первый экран сразу читается и не перегружен;
- мятные акценты цепляют важные фразы, но не выглядят пёстро;
- на телефоне легко читать заголовки, карточки и верхнюю навигацию.

## Известные ограничения

- проект пока полностью статический, без формы, аналитики и каталога;
- шрифты зависят от `fonts.googleapis.com`;
- правая часть первого блока убрана временно и пока не заменена новым контентом.
